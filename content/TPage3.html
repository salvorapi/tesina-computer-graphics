Caratteristiche di un file grafico

Cos'è un file grafico

Un file grafico è un file contenente dati grafici, cioè dati che permettono do costruire un'immagine. Questi file hanno lo scopo di conservare, organizzare e recuperare dati grafici in modo efficiente e logico. L'implementazione di un file grafico è molto complessa, ma anche importante per la velocità con cui può essere letto, per lo spazio che occupa e per la semplicità con cui può essere utilizzato in un'applicazione. Vedremo successivamente l'organizzazione generale di un file grafico.

Header e Palette

L' Header, o testata di un file grafico, è una serie di byte contenenti informazioni come l'altezza, la lunghezza, la versione del formato , la profondità dell'immagine da visualizzare. Genericamente, i primi byte dell'header vengono chiamati Magic Number e contengono l'informazione sul tipo di formato. A volte l'header contiene un campo puntatore che contiene un puntatore ad un secondo header. La dimensione in byte dell'header dipende dal tipo di formato grafico. La palette, detta anche mappa dei colori, si trova subito dopo l'header e può essere di lunghezza variabile. La palette è una componente fondamentale del file grafico, in quanto contiene l'indice dei colori dell'immagine da visualizzare. Un colore viene rappresentato con 3 byte, che corrispondono rispettivamente al rosso, verde e il blu.
